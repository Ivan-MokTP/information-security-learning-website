!!!This website is a school project and it connects to Polyu's database. Therefore, it requires connection to Polyu's WiFi in order to connect to database.!!!

Requirement:
1. Nodejs [v10.15.2]
2. Npm [v6.4.1]

Steps:
1. Install NodeJs.
1.1 During NodeJs installation, Npm will be installed as well

2. In the cmd, go to the project directory
3. Type command "npm install". It will install all necessaey packages.
